مدیریت وابستگی‌ها در Docker Compose

اگر چند سرویس را با Docker Compose بالا می‌آوری، فقط ترتیب اجرا کافی نیست. این مقاله تفاوت بین depends_on، healthcheck و wait-for-it را توضیح می‌دهد تا سرویس‌ها دقیقاً وقتی آماده‌اند بالا بیایند.

  • چرا start شدن کانتینر با ready بودن سرویس فرق دارد
  • بهترین روش برای صبر کردن تا آماده شدن دیتابیس
  • نمونه‌های واقعی برای Compose و API

استفاده از Profiles در Docker Compose
